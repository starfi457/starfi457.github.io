4. Classe Veiculo

• Problema: Crie uma classe Veiculo com os atributos: o marca (string), o modelo (string), o ano (número). Adicione métodos: o descrever(): retorna uma string descrevendo o veículo, como "Marca: Ford, Modelo: Fiesta, Ano: 2020". • Crie uma classe Carro que herde de Veiculo e adicione: o portas (número). O método descrever() deve incluir o número de portas. • Extra: Crie uma subclasse Moto que herde de Veiculo e inclua: o cilindradas (número). Modifique o método descrever() para incluir esse atributo.

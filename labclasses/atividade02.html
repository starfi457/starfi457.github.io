2. Classe ContaBancaria

• Problema: Crie uma classe ContaBancaria com os seguintes atributos: o titular (string), o saldo (número). Adicione os métodos: o depositar(valor): aumenta o saldo. o sacar(valor): diminui o saldo se houver saldo suficiente; caso contrário, exiba uma mensagem de erro. o mostrarSaldo(): exibe o saldo atual. • Extra: o Crie uma classe ContaCorrente que herde de ContaBancaria e adicione um atributo limite (número). o Modifique o método sacar() para permitir saque até o valor do limite.

Laboratório de CRUD

Desenvolveremos um sistema simples para gerenciamento de livros para uma biblioteca. O sistema deve permitir o Cadastro, Leitura, Atualização e Exclusão (CRUD) de informações sobre os livros utilizando classes e objetos em JavaScript.
